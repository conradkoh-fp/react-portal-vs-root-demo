CreatePortal vs CreateRoot

This demo aims to show the benefits of createPortal used to access data rendered outside of the app root.
Aims:

  1. Demonstrate context state access
  2. Demonstrate reactivity
How to verify:
  1. Start the app
  2. Click on "Show"
  3. Validate the following:
    • The one using createRoot fails to access the state
    • The one using createPortal successfully accesses the state and maintains reactivity
